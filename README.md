## Cara Run
Program ini sudah dikonfigurasi lewat `Makefile`. 

jalankan perintah berikut untuk build dan langsung menjalankan program:

```bash
make run
```

jalankan perinta berikut untuk menghapus file hasil build:

```bash
make clean
```
